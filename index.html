About

Hi, I'm Ashly — a web design student and creator.

This website is about web design created by Ashly Rico as a student project for ARTD 245: Web Design at North Central College in Naperville, IL, in 2025.

All the text comes from articles located on the Dribbble website.All the images were created by me using Photoshop and Canva. The website has been created using HTML and CSS in Dreamweaver.

Article Sources

  • Ultimate Guide to Choosing Colors For Web Design — Stephanie Corrigan
  • The Ultimate Guide to Choosing Fonts — Monica Galvan
  • 11 Principles of Design (and How To Use Them) — Renee Fleck

Project Brief

Design and develop a visually pleasing, interactive website with multiple HTML pages, navigation connecting all the pages, featuring a cohesive color scheme, a clear visual hierarchy, a balanced layout, exciting visuals, and text. The goal is to learn how to code/develop an interactive website by applying design principles.
